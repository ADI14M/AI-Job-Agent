# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-02
### Added
- Fully automated demo orchestration (`demo.sh` and `demo.py`).
- Fallback mock parsing mechanisms for offline demonstration.
- Application Decision Engine (ATS Scorer, Skill Gap Analyzer, Resume Optimizer, Cover Letter Generator).
- Playwright-based autonomous ATS submittion engine.
- Persistent Career Memory event tracking module.
- AI Provider integration directly tied to local Ollama.
- ChromaDB persistence backend for fast resume and job vector similarity look-ups.
- Full architectural diagram generation.
